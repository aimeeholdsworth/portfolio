Migraine Tracking and Visualisation

For my final year project I created a web application to track and visualise user entered data for migraines. I created the application using HTML and Javascript, specifically Node.js and Chart.js. The project combined UX and technical skills to create something that would work well for the users needs and works with real world data through an API and the user entered data.
